Explanations

This example tracks almost any car's rim (it requires to have some contrast between rim and tire). You'll see the rim's outline projected into the camera view. The algorithm will use this very basic shape to initialize If you roughly align it with the rim, it'll instantly snap. After snapping took place you'll see the augmentation placed over the rim. The tracking configuration has been created using metaio Creator's "Object tracking / 3D CAD Model (beta)."
