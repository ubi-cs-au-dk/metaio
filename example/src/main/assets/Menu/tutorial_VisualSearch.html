Introduction

This example shows how to perform visual search with metaio SDK that is based on the metaio visual search server. Simply put, all the heavy lifting is handled on the server side, which alleviates the burden of user devices. On the client side, camera images are sent to the server in an attempt to find a match. The server responds with the visual search result, which provides the tracking information and the filename of the image if visual search is successful, or an error message if it fails.
